4 Global Goals

Description: This project showcases 4 of the 17 global goals in a small, standard picture made with code.

Concepts Learned: Making pictures with code, including using loops and variables for graphics.

Challenges: It was time-consuming, as I had to carefully plan and code each goal's image.

Code:


### Life Below Water TANAY
### Rectangle for the goals we chose
Rect(0,0,200,200,fill='blue',border='black')

### Labelling so it is clear what goal we have done
Label('Life Below Water',100,20,size=15,bold=True,font='monospace')

### Drawing the Fish
Oval(140,80,70,35,fill=gradient('cyan','blue'),border='black')
Polygon(120,80,80,60,80,100,fill=gradient('cyan','blue'),border='black')
Oval(90,150,80,35,fill=gradient('cyan','blue'),border='black')
Polygon(60,150,30,130,30,170,fill=gradient('cyan','blue'),border='black')

### Drawing the bubbles for it to be a bit more realistic in a water environment.
Circle(150,120,10,fill='lightCyan',border='black')
Circle(170,50,10,fill='lightCyan',border='black')
Circle(50,100,10,fill='lightCyan',border='black')
Circle(30,60,10,fill='lightCyan',border='black')

Explanation: In this code, you can see that I have used the programming language Python, with various shapes and some comments. In this code, you can see labels, which are used to label the part of the project-- Life Below Water. We have also used various shapes to help us layer and make the fish, and bubbles, using the gradients to express the color shown in the picture.
