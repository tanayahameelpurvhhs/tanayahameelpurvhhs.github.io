Life on Land Slides

Description: This interactive slideshow project presents UN Goal 15, Life on Land, which shows us how the land is being degraded by multiple factors.

Concepts Learned: Learning functions, Parameters, event handlers such as onMouseRelease(), and muliple other things related to this project.

Challenges: Making transitions between slides work properly and positioning shapes responsively were tricky parts of the project.

Code:


Rect(0,0,400,400,fill='skyBlue')

sun = Star(120,150,20,9,roundness = 90, fill='yellow')

    sun.centerX = x
    sun.centerY = y
    sun.radius = radius
    sunback.centerX = x
    sunback.centerY = y
    sunback.radius = (radius * 3)/2
    
tree = Polygon(100,345,140,315,155,280,195,280,210,315,250,345,fill='saddleBrown')

def drawLogoB(x,y,size):
    tree.centerX = x
    tree.centerY = y
    tree.width = size + 40
    tree.height = size

These are drawn all across the slides, to an effect of Life on Land, and what Life on Land depends on, being the tree stump, and the sun respectively. Clean The Water!

Description: A clicking game where you clean up water by clicking on the tank. Each time you click on the tank, the water gets cleaner. It supports Global Goal 6: Clean Water and Sanitation.

Concepts Learned: I learned how to use mouse events, variables, conditionals, and counters to create interactive gameplay with resetting behavior.

Challenges: Trying to transition from the next slide and making it so that the tank goes down in pollution required debugging and testing.

Code:


def cleanReplica(opacityNumber,value):
    app.cleanLevel+=1
    newOpacity = tank.opacity-opacityNumber
    
    if newOpacity < 0:
        click2start.value='Round ' + (app.slide + 1)
    else: 
        tank.opacity = newOpacity
        if tank.opacity > 70: 
            click2start.value = 'Game ON!'
        elif tank.opacity > 50 :
            click2start.value = 'Keep it up'
        elif tank.opacity > 30:
            click2start.value = 'Crossed halfway mark!'
        elif tank.opacity > 0:
            click2start.value = 'Almost There'
        else:
            click2start.value='Good Job! You did it!'

    print(app.cleanLevel)
    print(tank.opacity)

        
def startGame(x,y):
    infoBackground.visible=False
    base.visible=True
    title.visible=True
    info1.visible=False
    info2.visible=False
    info3.visible=False
    info4.visible=False
    click2start.visible=True
    board.visible=False
    tank.visible=True
    app.cleanLevel=0
    tank.fill=gradient('darkGreen','green','darkGreen')
    app.gameStarted = True

def onMousePress(x,y):
    if app.gameStarted == False:
        startGame(100,100)
    else:
        if app.slide == 0 :
            cleanReplica(20,100)
        elif app.slide == 1:
            cleanReplica(10,100)
        elif app.slide == 2:
            cleanReplica(5,100)

Explanation: This function checks for pollution clicks. Once an (n) amount of clicks are reached, the clean tank fills and the pollution is removed from the tank, reinforcing the cleanup idea.

UN Goal 12 Quiz

Description: This is a quiz my partner and I made to teach about UN Goal 12 (Responsible Consumption and Production). It includes mostly true/false and multiple-choice questions about recycling and sustainability.

Concepts Learned: I learned how to use `hits`, `contains`, and `onKeyPress` events. We designed the quiz like a slideshow that updates as users interact.

Challenges: Teamwork was a major challenge at first, which delayed our progress. But once we got organized, we pushed through together.

Code:


def onMousePress(mouseX, mouseY):
    if (Option1.hits(mouseX, mouseY)):
        if (Option1Label.value == app.Question1):
            app.score += 1
            Option1.fill = 'yellow'
        else:
            app.score -= 1
    elif (Option2.hits(mouseX, mouseY)):
        if (Option2Label.value == app.Question1):
            app.score += 1
            Option2.fill = 'yellow'
        else:
            app.score -= 1
# (Continues for Option3-6)...

Explanation: This function checks which quiz option the user clicks and updates the score based on whether the selected answer is correct. We also change the fill color to give visual feedback.
